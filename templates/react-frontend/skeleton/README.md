   ```bash
   npm run dev
   ```

3. **Open your browser:**

### Building for Production

```bash
npm run build
```

### Docker Deployment

1. **Build Docker image:**
   ```bash
   docker build -t ${{ values.name }} .
   ```

2. **Run container:**
   ```bash
   docker run -p 80:80 -e VITE_API_URL="${{ values.apiUrl }}" ${{ values.name }}
   ```

{% if values.deployToKubernetes %}
### Kubernetes Deployment

1. **Apply Kubernetes manifests:**
   ```bash
   kubectl apply -f k8s/
   ```

2. **Check deployment status:**
   ```bash
   kubectl get pods -l app=${{ values.name }}
   ```

3. **Access the application:**
   - Local: `http://${{ values.name }}.local`
   - Port-forward: `kubectl port-forward svc/${{ values.name }}-service 8080:80`
